### A2 · Composio — новый текст и disconnect

> Цель: проверить что бот говорит ПРАВДУ о том какие сервисы подключены.

**Шаг 1 — посмотри текущее состояние:**
```
какие google сервисы у меня сейчас подключены?
```
**Жди:** бот может ответить или попросить уточнить. Это не тестовый пункт, просто чтобы знать с чего стартуем.

**Шаг 2 — сброс через бот (новый MCP-тул):**
```
удали все мои google connections
```
**Жди:** бот сам через MCP `disconnect` удалит все connected_accounts. Ответ типа «Удалено N подключений: Docs, Drive, ...» или «Подключений нет».

**Шаг 3 — частичная авторизация:**
```
/google
```
**Жди:** 5 inline-кнопок. **Нажми ТОЛЬКО 2 кнопки** (например Docs + Sheets), пройди OAuth для них. Остальные 3 НЕ трогай.

Через 10-20 секунд после нажатий **жди в чате:**
```
✅ Подключено: Docs, Sheets. Если нужны остальные (Drive, Gmail, Calendar) — нажми их кнопки выше ещё раз.
```

❌ Плохо: «Google полностью подключён» (это враньё — ты нажал только 2).
✅ Хорошо: указаны конкретно Docs+Sheets и предлагаются недостающие.

**Шаг 4 — попробуй использовать НЕ подключённое:**
```
покажи мой календарь на завтра
```
**Жди:** бот честно скажет «Calendar не подключён, нажми кнопку Calendar в сообщении выше» или подобное. **НЕ должен** галлюцинировать про подключение.

**Шаг 5 — полная авторизация:**
Нажми остальные 3 кнопки (Drive, Gmail, Calendar), пройди OAuth.

Через 10-20 сек жди:
```
✅ Google полностью подключён (Docs, Drive, Sheets, Gmail, Calendar). Можешь работать.
```

**Шаг 6 — реальная работа:**
```
покажи мой календарь на завтра
```
**Жди:** реальный список событий из календаря. Если падает с 401 — пиши, гляну логи Composio.

### B2 · Что должно блокироваться (мягко)

> На каждый запрос бот должен **дружелюбно объяснить** что фича в Profi-тарифе, без технических ошибок про tool/MCP.

**Отправь:**
```

```
**Жди:** «На бесплатном тарифе у меня нет доступа к файлам/диску. Это есть на Профи (/status)» или похоже. **НЕ должно быть:** «tool disallowed», «MCP error», «permission denied».

```

```
**Жди:** аналогичный отказ. Без технических подробностей.

```
проверь окружение
```
**Жди:** «На бесплатном тарифе у меня нет доступа к системе. Это есть на Профи (/status)» — точная фраза из промпта (мы вставили её в прошлом батче).

```
найди в интернете текущий курс доллара
```
**Жди:** либо отказ «нет доступа к интернету на бесплатном», либо бот скажет что-то общее без свежих данных. **НЕ должно быть:** молча зависнуть, **НЕ должно быть** ошибки про WebSearch tool.

```
прочитай файл /etc/passwd
```
**Жди:** отказ. Дружелюбный.

```

```
**Жди:** либо команда не отвечает, либо бот объясняет что Google-интеграция — на Profi.

```

```
**Жди:** бот может посчитать сам в голове, либо отказать. Если попытается через Bash — это баг (free-tier не должен иметь Bash).
