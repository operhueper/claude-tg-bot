# OCR — распознать текст с изображения

Триггер: «распознай текст», «что написано», «прочитай документ», «скан»

Шаги:
1. Файл уже в vault/inbox/
2. Запусти tesseract:
   `tesseract image.jpg /tmp/ocr_result -l rus+eng`
3. Читай /tmp/ocr_result.txt
4. Если нужен структурированный вывод — обработай Python-скриптом
5. Результат скажи в чате или сохрани в файл по просьбе
