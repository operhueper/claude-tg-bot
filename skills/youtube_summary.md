# YouTube → краткое содержание

Триггер: ссылка youtube.com/watch?v=... + «перескажи», «о чём», «суммаризируй»

Шаги:
1. Скачай субтитры: `yt-dlp --write-auto-sub --skip-download -o /tmp/yt URL`
2. Если нет субтитров — скачай аудио и транскрибируй через whisper (если доступен)
3. Прочитай .vtt/.srt файл, удали тайм-коды
4. Суммаризируй текст: выдели главные темы, ключевые моменты, выводы
5. Ответь в чате структурированным резюме
